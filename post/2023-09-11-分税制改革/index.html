分税制改革

Posted by zhangtao on Sunday, September 10, 2023

在上世纪90年代，由于国内外的多重危机，中央财政陷入了严重的困境，甚至不得不向地方借款的情况屡见不鲜。

朱镕基上任总理后，果断启动了大规模改革，着手实施中央分税制财政改革。

所谓分税制，就是将各类税收进行详细分类，有些归属于地方，有些则属于中央，还有一部分是中央和地方共同分享的税收。例如，增值税的分成比例变为地方占25%，中央占75%。

这项改革在实施的第一年就取得了显著成效，中央政府获得了更多的财政资金，从而能够进行宏观经济调控，并推动了共同富裕的改革。此外，这也为后来明确规定军队不得从事商业活动创造了条件（在中央财政困难时，曾允许军队从事商业活动以弥补开支）。

然而，在改革过程中，地方政府面临着巨大的阻力。朱总理亲自走访各地，最终才得以推动改革的实施。

这一改革带来了积极和消极的影响。后来，地方政府的财政收入减少，因此开始通过土地出售来填补财政亏空，这导致了房价的上涨。

总的来说，这一历史过程可以概括为：

中央统一管理 => 地方包干 => 分税制

原本中央提供财政支持 => 地方拥有财政自主性，中央财政紧张 => 共同富裕的追求
